## 0. 一句话项目定位

面向印尼市场的 B2C 跨境电商独立站,三语展示(印尼/中/英,印尼默认),印尼盾唯一结算,会员制下单(必须注册才能下单),PC+手机+平板全响应式,完整后台运营 + 客服 + 数据统计 + 财务独立管理 + 优惠券营销。

## 2. 基础全局规范(全站硬性强制)

### 2.1 终端适配
- 全站响应式,完美适配 **PC + 手机 Web + 平板**
- 三端数据实时同步,体验统一
- 触摸热区 ≥ 44px,无错乱/挤压/失效

### 2.2 多语言(范围严格)
- ✅ **支持三语**:导航、按钮、提示语、弹窗、系统字段、后台固定文字
- ❌ **不支持三语**:商品、分类、广告、详情、参数等用户编辑内容
- 默认首选:**印尼语(id)**
- 路由:`/id /zh /en`,`id` 隐藏默认走 `/`

### 2.3 货币
- 全站唯一:**印尼盾(IDR)**
- 展示格式:`Rp 1.234.567`(`Intl.NumberFormat('id-ID')`)
- 无币种切换,无残留

## 3. 前端用户端功能

### 3.1 浏览与下单权限管控(强制锁单)
- **游客**:仅可浏览首页、商品列表、商品详情、分类、广告
- **游客禁止**:加入购物车、下单、提交订单
- **未登录访问下单页**:自动跳登录,登录后回原页面
- 游客**可看价格**(提升转化,只锁下单)

### 3.2 会员注册
- **3 项必填**:邮箱 + 印尼电话号码 + 确认密码
- **印尼手机号格式**:`+62xxxxxxxxxx`(只做格式校验,**不验证真实性,不接验证码**)
- **邮箱格式校验**(做格式校验,**不接邮件验证链接**)
- 密码复杂度:至少 8 位 + 字母数字混合
- 注册后自动登录,跳会员中心

### 3.3 会员中心(7 项数据)
- 账户余额
- 订单付款状态(待付款/凭证已提交/审核通过/待发货/运输中/已完成/已取消)
- 待发货、待收货(基于订单状态)
- 评价记录(可发布图文评价)
- 退换货记录
- 全部历史购买记录
- 余额变动明细 + 审核状态

### 3.4 商品展示
- **9 张主图**(本地上传或超链接双模式)
- 详情页图文
- **短视频**(本地上传或超链接)
- 全自定义参数(无数量/类型限制)
- 前台展示后台设置的三种排序:**手动 / 流量 / 点击率**

### 3.5 订单与支付(唯一:银行线下对公转账)
- 全站**无 COD**,**无第三方支付网关**
- 结算页显示支付方式:**银行转账**
- 提交订单后 → 订单详情页显示对公账户(开户行/账户名/账号)
- 用户转账后,在订单详情页**上传转账凭证**(本地上传或超链接双模式)
- 状态自动流转到"凭证已提交",等后台审核
- 前台实时显示审核状态(待审核/审核通过/审核驳回)
- 凭证被驳回后,用户**可重新上传**(最多 3 次),订单状态回到"待付款(待重传)"

### 3.6 首页广告
- 两类广告:**顶部横幅** + **全局弹窗**
- 严格跟随后台设置(启停/排序/定时)
- 后台修改后前台实时生效

### 3.7 站内客服
- 全站右下角浮窗
- **非工作时间显示"留言入口"**(非关闭,留言转站内通知)
- 客服可发送**站内商品链接**(点击跳商品详情)

### 3.8 商品评价
- 订单状态完结后自动开放评价入口
- 支持**图文评价**
- 评价先审后公(管理员后台审核通过才公开)
- 合规评价在商品详情页公开展示

### 3.9 优惠券
- 会员中心可领取优惠券
- 下单结算可选 1 张抵扣
- 未满足门槛的券**自动置灰**不可选
- 单订单**仅 1 张**,不可叠加

## 4. 后端管理系统功能

### 4.1 权限账号体系(主账号 + 子账号)

**超级主账号(超管)**:
- 100% 最高权限
- **财务超管角色**:可直改会员余额(不需审核)
- 拥有最终审核权限
- 拥有操作日志查看权限

**子账号**:
- 7 大模块独立授权(可拆分到按钮级)
- 无最高级审核权限
- **无日志操作/查看权限**
- 无操作日志的删除/查看权限(零入口)

**7 大模块**(可独立分配):
1. 商品管理
2. 分类管理
3. 订单与支付
4. 广告营销
5. 站内客服
6. 站点设置
7. 数据统计 / 财务管理(可拆分)

### 4.2 全站操作日志(强制锁死)
- **永久留存**:**任何人、任何账号、任何角色均无法删除/修改/清空/覆盖**
- 系统**不提供任何日志管理入口**(无 API、无 UI、无 CLI)
- 记录字段:操作账号、操作时间、操作模块、操作内容、**修改前后数据对比**、设备信息
- 覆盖操作:增/删/改/状态调整/余额变动/参数配置/审核操作/优惠券/会员数据修改
- 不可篡改,永久可追溯

### 4.3 会员与评价管理
- 会员列表:邮箱、印尼手机号、注册时间、账户余额
- 会员详情:订单/物流/评价/退换货/购买记录/余额变动
- 评价管控:可编辑/删除评价

### 4.4 商品管理
- 9 张主图(本地上传/超链接双模式)
- 详情图文 + 短视频
- 全自定义参数
- 三种排序模式(手动/流量/点击率)

### 4.5 分类管理
- 完全自定义 + 多级层级
- 图标(本地上传/超链接双模式)
- 排序 + 简介

### 4.6 订单与支付
- **唯一支付**:银行对公转账(后台可配置多个对公账户)
- **转账凭证审核**:用户上传凭证后,后台审核通过/驳回(可填备注)
- 多 SKU 多仓库
- **多物流单号**:同一订单可录入多个快递单号
- **自定义物流方式**:管理员手动自由输入物流公司名称(无固定模板)
- 状态流:待付款(待转账)→ 凭证已提交 → 审核通过(已到账)→ 待发货 → 运输中 → 已完成 → 已取消 / 退换货
- 订单超时未付款:24h 自动取消

### 4.7 广告营销
- 横幅 + 弹窗
- 新增/编辑/删除/排序
- 一键启停 + 定时上下线
- 图片(本地上传/超链接双模式)

### 4.8 站内客服后台
- 独立聊天后台
- 可发**站内商品链接**(一键插入商品卡片)
- 可设置**工作时间**

### 4.9 站点基础配置
- LOGO / 全站图标 / 前台固定文字(后台可视化修改)
- 图片(本地上传/超链接双模式)

### 4.10 多维数据统计
- 核心 KPI:总交易额、总交易笔数、平均客单价
- 每日访客 + 访客趋势图
- 销售趋势 + 热销商品排行
- 时间筛选:**按天 / 按周 / 按月 / 按年**

### 4.11 独立财务管理板块

**专属独立模块**,与订单/销售数据**独立区分**。

**会员余额管控**:
- **普通子账号**:仅可**提交余额修改申请**(增加/扣除/重置),需超管审核
- **财务超管**:直接修改余额(不需审核)
- 三种状态:**待审核 / 审核通过 / 审核驳回**
- 余额变动记录:提交账号、审核账号、变动时间、变动金额、修改前后余额、备注、设备信息
- 会员中心同步展示自身余额变动 + 审核状态

**财务数据**:
- 按天/周/月/年筛选
- **一键导出 Excel**(通用表格)
- 实时同步,无延迟

### 4.12 优惠券系统
- 后台新增/发布/编辑/下架/删除
- 双模式:**固定金额 / 百分比折扣**
- 自定义:名称、优惠额度、使用门槛、有效期(固定起止 OR 有效天数)、发放总量、单人领取次数
- 状态:未使用 / 已使用 / 已过期 / 已下架
- 单订单限 1 张,门槛未达置灰
- 所有操作同步记录操作日志
- 到期自动失效

## 5. 4 个风险解决方案(老板已拍板)

### 风险 1:转账凭证人工审核
- **全部人工审核**
- 推送通道:**只用站内红点**(不接邮件/WhatsApp/短信)
- 后台首页 / 订单列表加"待审核"红色未读数
- SLA 目标:4 小时内审完

### 风险 2:短信验证付费
- **不接任何验证码**(WhatsApp/短信/邮件都不接)
- 手机号只做格式校验(`+62xxxxxxxxxx`)
- 邮箱只做格式校验
- 无短信/邮件/WhatsApp 验证流程,注册即可登录

### 风险 3:财务二次审核流程重
- **新增"财务超管"角色**
- 财务超管:**直接修改会员余额**,无需审核
- 普通子账号:仍走超管审核(子账号仅能"申请修改")
- 超管可批量审核(批量通过/批量驳回)

### 风险 4:WebSocket 在印尼网络不稳定
- **方案 A:消息入库 + 30s 心跳 + 自动重连 + 5s 轮询 fallback**
- 真实数据存 DB,WebSocket 只推送通知
- 断线自动重连,带 `last_msg_id` 增量拉取
- 连续失败 3 次 → 自动降级为 5s 轮询
- 网络恢复自动升回 WebSocket

## 6. 关键功能补充细节(0.2 文档延伸)

| 细节 | 决定 |
|------|------|

| 游客能否看价 | 能看(只锁下单,不锁价格) |
| 评价发布 | 先审后公(管理员后台审核通过才公开) |
| 客服非工作时间 | 显示"留言入口",留言转站内通知客服 |
| 订单超时 | 24h 未付款自动取消 |
| 商品列表分页 | 20 条/页 |
| 图片/视频大小 | 图片 ≤ 5MB,视频 ≤ 50MB |
| 操作日志保留 | 永久(印尼 UU PDP 倾向 5 年+,按"永久"实现) |
| 注册时勾选同意 | 是(《服务条款》《隐私政策》) |
| 转账凭证人工审 | 是,全部人工(站内红点提醒) |
